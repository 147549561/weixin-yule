# 安装
> 下载微信开发者工具 https://mp.weixin.qq.com/debug/wxadoc/dev/devtools/download.html

> 克隆项目  git clone https://github.com/aymfx/mylittleblog.git

>将项目导入微信开发者工具 在配置中勾选开发请求不校验域名
